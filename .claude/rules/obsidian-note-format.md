# Obsidian Note Format Rules

Two note types exist in this project: **Exam Notes** (primary) and **Raw Transcript Notes** (intermediate).

---

## 1. Exam Note (per Bài) — Primary Format

Generated by `write_exam_notes.py`. One file per video lesson.

### YAML Frontmatter

```yaml
---
title: "<Lecture> - Bài N: <Title>"
tags:
  - <course-slug>     # e.g. word, excel, powerpoint
  - cntt-co-ban
  - exam-note
---
```

### Structure

```
# Bài N: <Title>

← [[N-1 - PrevTitle|Bài N-1]]  |  [[Lecture|📋 Mục lục]]  |  [[N+1 - NextTitle|Bài N+1]] →

---

### <Section heading>

`Menu → Submenu → Command`   ← step in backtick code span

| Column A | Column B |        ← table for memorizable values
|----------|----------|
| ...      | ...      |

> **Nhớ nhanh: <rule of thumb>**   ← bold blockquote for key mnemonics

⚠️ <one-line warning for the most common exam mistake>

---

## Ghi Chú Cá Nhân

> [!question] Chưa hiểu
> -

> [!check] Đã nắm
> -

---

← [[...]] | [[...]] | [[...]] →
```

### Content rules

- **No prose, no transcript text** — steps and values only
- Each lesson must fit in **under 80 lines**
- Use `Menu → Path → Command` format for all UI steps (backtick code span)
- Bold the single most important value in each table: `**3 cm**`
- One `⚠️` per note maximum — the single most-tested gotcha
- Keyboard shortcuts go in a `| Phím | Chức năng |` table

## 2. Raw Transcript Note (intermediate)

Generated by `build_notes.py`. Replaced by exam notes after `/exam-notes` is run.

### YAML Frontmatter

```yaml
---
title: "<Lecture> - Bài N: <Title>"
tags:
  - <course-slug>
  - cntt-co-ban
  - <content-slug>    # ascii kebab-case from Vietnamese title
source: "<YouTube URL>"
---
```

### Transcript content rules

- Break every ~40 words at a natural Vietnamese word boundary
- Separate paragraphs with blank lines
- Always precede with:
  `> *Bản ghi được tổng hợp từ phụ đề tự động của video. Một số từ có thể bị nhận dạng sai do giọng nói.*`

## 3. Index Note

One per lecture folder. Filename = lecture name exactly.

```
# <Lecture Name> — <Course subtitle>

> 📺 [YouTube Playlist](<url>)
> 🗂️ <N> bài

---

## Mục Lục

| Bài | Chủ đề |
|-----|--------|
| [[Lecture Bài 1 - Title\|Bài 1]] | Title |
...

---

## Checklist Học Tập

- [ ] Bài 1 — Title
...

---

> [!tip] Mẹo học hiệu quả
> Đọc note → Thực hành trong Word → Tick checklist
```

---

## Navigation Links

Format: `← [[N-1 - PrevTitle|Bài N-1]]  |  [[Lecture|📋 Mục lục]]  |  [[N+1 - NextTitle|Bài N+1]] →`

- First note: omit ← link
- Last note: omit → link
- Filename and wikilink target must match **exactly** (including spaces and diacritics)

## Callout Types

| Callout | Use |
|---------|-----|
| `[!question]` | Things not yet understood |
| `[!check]` | Concepts confirmed understood |
| `[!tip]` | Study / exam tips |
| `[!note]` | Content clarifications |
| `[!info]` | Source / metadata |
